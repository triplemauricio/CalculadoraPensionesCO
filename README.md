# Simulador Pensional Colombia 2026 🇨🇴

Herramienta gratuita de proyección actuarial para el sistema pensional colombiano.

## Marco Legal
- Ley 100/1993 (Régimen de Transición ISS)
- Ley 2381/2024 (Reforma de Pilares)
- Sentencia C-197/2023 (Reducción gradual semanas mujeres)
- Ley 797/2003

## Ejecutar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy en Streamlit Cloud
1. Sube este repositorio a GitHub
2. Ve a [share.streamlit.io](https://share.streamlit.io)
3. Conecta tu cuenta de GitHub
4. Selecciona el repositorio y `app.py`
5. Click en **Deploy**
